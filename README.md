# 亲爱的唐荣宝宝的专属页面

这是一个为亲爱的唐荣宝宝创建的浪漫互动页面。

## 项目文件结构

- `index.html` - 主页面结构
- `style.css` - 页面样式
- `script.js` - 交互功能
- `.gitignore` - Git忽略文件

## 如何部署到GitHub Pages

### 步骤1: 创建GitHub仓库
1. 登录GitHub账号
2. 点击右上角的"+"图标，选择"New repository"
3. 输入仓库名称（例如："zhiweini"）
4. 选择公开或私有仓库
5. 点击"Create repository"

### 步骤2: 上传文件
1. 在仓库页面，点击"Upload files"
2. 拖拽所有文件（index.html, style.css, script.js, .gitignore, README.md）到上传区域
3. 添加提交信息，例如："Initial commit"
4. 点击"Commit changes"

### 步骤3: 启用GitHub Pages
1. 进入仓库的"Settings"页面
2. 向下滚动到"GitHub Pages"部分
3. 在"Source"下拉菜单中选择"main"或"master"分支
4. 点击"Save"
5. 刷新页面，GitHub会显示您的网站URL（通常是 `https://[username].github.io/[repository-name]/`）

### 步骤4: 访问您的网站
稍等几分钟后，您可以通过GitHub提供的URL访问您的网站。

## 注意事项

- 如果添加了自定义域名，请确保配置了DNS设置
- 对于添加本地照片，需要将照片上传到仓库中，并更新script.js中的photoUrls数组
- 网站支持响应式设计，可以在手机和电脑上正常显示

## 功能介绍

- 三个互动按钮：惊喜、好听的、小请求
- 浪漫语录自动更新
- 点击效果和动画
- 键盘彩蛋：按L键触发爱心雨效果
